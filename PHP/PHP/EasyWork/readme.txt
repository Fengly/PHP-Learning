『软件简介』
      是九五时代最新推出的一款专门针对中小企业提高工作效率而生的专业的项目管理系统，系统内置任务管理、项目管理、日志管理、文档管理、公告管理、用户管理、组别管理、部门管理等功能，本系统最强大的就是他的项目控制流程和无限制任务分级功能，然您面对再复杂的项目都是随心所欲，同时还有强大的进度显示特效，然您快速而清晰的了解项目当前的进度，所有项目将从此一目了然。

====================================================================================
『入门说明』
      使用EasyWork项目管理系统之前，您要先到我们的官方论坛学习使用说明，地址：http://www.d-winner.com/。如果需要二次开发EasyWork项目管理系统，您需要先学习：Thinkphp框架、EasyUI前端框架、Dream使用说明，如果您不了解这些，您将很难对系统进行开发。各个框架使用API文档在“API文档”栏目都里能找到。

====================================================================================
『安装说明』
      首次进入系统会自动跳转到install安装程序中，如果没有跳转，请手动进入Install/index.php。
      如果是更新系统，请解压包，将upload中的文件直接覆盖到根目录即可。
