『软件简介』
      Dream项目管理系统（又称：梦想项目管理系统、Bug管理系统）是一款基于PHP+Mysql开发的web基础系统，系统内置项目管理、Bug管理、出差日志、Bug百科、用户管理、组别管理、部门管理，同时系统还能动态开启子公司，当您的公司有多家分公司时，这个功能可方便您的操作。
      系统设计大部分采用动态修改模式，左侧菜单栏、联动数据、系统参数配置等都是采用后台添加修改，使用方便简单。
      内置了6套皮肤切换，邮件发送等功能，您无需再重新开发这些附加功能。数据加载基本采用json数据流，让系统更高效更稳定。
      Dream后端采用了ThinkPHP，前端采用EasyUI库等主流框架开发而成，内置强大的项目管理流程，普通使用者只需安装系统即可直接使用。中文api说明文档详细。对于开发者上手迅速，使用起来得心应手。

『安装说明』
      首次进入系统会自动跳转到install安装程序中，如果没有跳转，请手动进入Install/index.php。
      如果是更新系统，请解压包，将upload中的文件直接覆盖到根目录即可。

      如果出现安装完成后无法登陆时，请检查 dwin_user_table 数据表是否有数据，如果没有数据请手动添加一条数据，格式如下：
      username      realname      password                               email              status       access      ...
      admin         admin         21232f297a57a5a743894a0e4a801fc3       自己的邮箱         1            9999        后面的字段默认
